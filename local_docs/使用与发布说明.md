# 使用与发布说明

## 本地运行

```bash
swift run ConvertVideo2MP3
```

运行后点击“选择目录”，应用会递归扫描视频文件。默认会选中未成功转换的任务，可以手动单选、多选、全选或清空选择。

## 转换选项

- 并发数：可选 4、6、8。
- 成功后删除源视频：勾选后，本次转换成功并生成最终 `mp3` 后会删除对应源视频；如果 `mp3` 已经存在且源视频仍存在，会跳过转码并只补做删除源视频。
- 进度：表格会显示每个任务的 0~100% 进度，顶部也会显示总体进度。
- 文件大小：表格显示每行视频大小和 MP3 大小，顶部汇总当前扫描结果的视频总大小和 MP3 总大小。

## 播放快捷键

先点击表格中的任意一行，让表格获得焦点：

- `Space`：使用系统默认播放器播放该行源视频。
- `Ctrl+Space`：播放该行生成的 `mp3`。

如果 `mp3` 尚未生成，应用会提示“这个视频还没有转化完成，转化成功后可以播放”。如果 macOS 系统快捷键占用了 `Ctrl+Space`，需要先关闭输入法切换快捷键或改用 Finder 打开生成的 MP3。

## 清理未完成下载文件夹

点击“清理 .mp4.part 文件夹”后，应用会在当前根目录下递归查找包含 `.mp4.part` 文件的文件夹，并弹窗展示数量和路径预览。

确认后会递归删除这些文件夹。这个操作用于清理未完成下载残留，删除后不能通过应用撤销，所以应用会先弹窗确认。

## 构建、安装和打包

```bash
./scripts/build_release.sh
```

脚本会执行：

1. `swift test`
2. `swift build -c release`
3. 生成应用图标 `AppIcon.icns`
4. 生成 `dist/ConvertVideo2MP3.app`
5. 生成 `dist/ConvertVideo2MP3.zip`
6. 使用 ad-hoc codesign 签名
7. 自动更新到 `/Applications/ConvertVideo2MP3.app`

## 依赖

应用依赖本机 `ffmpeg` 和 `ffprobe`。如果没有安装：

```bash
brew install ffmpeg
```

当前实现会查找：

- `/opt/homebrew/bin/ffmpeg` 和 `/opt/homebrew/bin/ffprobe`
- `/usr/local/bin/ffmpeg` 和 `/usr/local/bin/ffprobe`
- `/usr/bin/ffmpeg` 和 `/usr/bin/ffprobe`

## 日志位置

日志文件位于：

```text
~/Library/Application Support/ConvertVideo2MP3/logs/app-yyyyMMdd.log
```

应用界面中也提供“打开日志”按钮。日志是结构化文本，例如：

```text
level=INFO event=conversion.started output=/path/a.mp3 source=/path/a.mp4 status=converting
level=DEBUG event=conversion.progress output=/path/a.mp3 progress=42 source=/path/a.mp4 status=converting
level=INFO event=conversion.succeeded output=/path/a.mp3 progress=100 source=/path/a.mp4 status=succeeded
level=INFO event=conversion.source_deleted output=/path/a.mp3 source=/path/a.mp4
level=INFO event=cleanup.part_folders.deleted folders=3 root=/path/root
level=ERROR event=conversion.failed error=... output=/path/b.mp3 source=/path/b.mov status=failed
```

## GitHub Release

如果仓库已经创建并登录了 `gh`：

```bash
git tag v1.0.0
git push origin main --tags
gh release create v1.0.0 dist/ConvertVideo2MP3.zip --title "ConvertVideo2MP3 v1.0.0" --notes "macOS desktop app for extracting MP3 audio from video files."
```

本次自动化会优先使用当前 `gh` 登录账号创建远程仓库并上传 Release 包。
